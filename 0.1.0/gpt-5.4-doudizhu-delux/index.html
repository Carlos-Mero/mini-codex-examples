斗地主规则简述

  • 54张牌，三人对战，17张手牌 + 3张底牌。
  • 叫分 0 到 3 分，最高分成为地主并获得底牌；若三家都不叫，则自动重发。
  • 支持单张、对子、三张、三带一、三带二、顺子、连对、飞机、飞机带牌、四带二、炸弹、王炸。
  • 炸弹和王炸会翻倍，春天会再次翻倍。
  • 支持提示、排序切换、快捷键操作与本地积分保存。
